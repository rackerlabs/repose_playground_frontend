Repose Test Instance

To evaluate Repose, you can select the filters you'd like to evaluate, plug in some data (a lot of required elements are filled in by default but you can update it as needed), and run requests against your configuration set. You can also then run performance tests to validate that the performance is up to your liking. Finally, you're able to download the configurations to plug them into your own Repose instance.
